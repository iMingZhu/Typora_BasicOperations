# 基础教程

## 1.标题[数个"#"+空格 前置]

```
# 一级标题
###### 六级标题
```

严格模式 手动补上空格 使markdown语法在其他文本编辑器上兼容



## 2.强调［用"**"或"__"包围]

```
**Hello world**
__Hello world__
```

或**选中文字**``ctrl``+``B``



##　3.斜体［用＂＊＂或＂＿＂包围］

```
*Hello world*
_Hello world_
```

或*选中文字*``ctrl``+``I``

(***斜体并强调***用"***"或"___"包围)

##　4.删除线[用"~~"包围]

```
~~Hello world~~
```

~~删除线~~



##　5.*高亮［用"=="包围]

**(注意:此为扩展语法)**

==高亮==



## 6.代码[用"`"包围]

```
`sudo rm -rf /*`
```

`print("hello")`



## 7.代码块[按三个"`"并回车]

````c
```
// 这里开始可以输入代码
#include <studio.h>
int main(){
    print("Hello world!\n");
    return 0;
}
```//(这三个"`"文本编辑器会帮你自动补全 一般不用手动输)
````

偏好设置里有``显示行号`` 重启Typora生效



## 8.引用[">"+空格 前置]

```
> 24岁,事学生
> > 学生特有的无处不在(恼)
```

引用可嵌套

E.G.

> 引用1
>
> > 引用2
> >
> > > 引用3



## 9.无序列表["-"或"+"+空格 前置]

```
-一棵是枣树
+另一棵还是枣树
*(不常用)
```

敲回车自动补全,敲回车后按下``Tab``会缩进一级

E.G.

- 发呆改变世界
  - 按下`Tab`变成子集



## 10.有序列表[数字+"."+空格 前置]

```
10. homo特有的
9.10>9
```

敲回车自动补全,敲回车后按下``Tab``会缩进一级

E.G.

我来这里就为了3件事:

1. 公平
2. 公平
3. 还是tm的公平!



## 11.*上标[用"^"包围]

**(注意:此为扩展语法)**

```
C语言中int的上限是2^31^-1=2147483637
```

E.G.

C语言中`int`的上限是2^31^-1=2147483637

 

##　12.*下标[用"~"包围]

**(注意:此为扩展语法)**

```
H~2~O是剧毒的
```

E.G.

H~2~O是剧毒的



##　13.*注释["[^]" 后置]

**(注意:此为扩展语法)**

```
> 引用1 注释1[^1]

[^1]:沃.兹基.硕德
```

需在文末写上注释对应内容

E.G.

> 引用1 注释1[^1]

[^1]:沃.兹基.硕德

a[^2]

[^2]:b



## 14.链接[常用"[]"+"()"分别包围文本与链接]

**(注意:文内跳转为扩展用法)**

```
[CS自学教程](https://csdiy.wiki)
[12.下标](#12.*下标[用"~"包围])
```

支持网页链接与文内跳转,按住`Ctrl`并`单击鼠标左键`即可跳转

E.G.

[CS自学教程](https://csdiy.wiki)
[12.下标](#12.*下标[用"~"包围])



## 15.任务列表["- [ ]"+空格 前置]

```
Todolist:
- [ ] 写代码
- [ ] 收拾行李
- [x] 做笔记
```

用`x`代替`[ ]`中的空格来勾选任务列表.在Typora中可以直接用鼠标左键单机勾选框

E.G.

Todolist:
- [ ] 写代码
- [ ] 收拾行李
- [x] 做笔记



## 16.表格[用"|"绘制表格边框]

```
|学号|姓名|年龄|
| :--- | :---: | ---: |(引号位置代表 左对齐 居中 右对齐)
|114514|田所|24|
|1919810|浩三|25|
```

第一行为表头,并由第二行分割线决定对齐方式与长度,第三行之后为表格数据

E.G.

|学号|姓名|年龄|
| :--- | :-: | ---: |
|114514|田所|24|
|1919810|浩三|25|



## 17.图片[直接拖或复制粘贴]

```
![图片](图片的位置)
```



## 18.分割线[按三个"*"或"-"或"_"并敲回车]

```
***
---(常用)
___
///(三个及以上都可)
```

("+"也可?)

由于`*`和`+`都会自动补全,所以`-`最方便

E.G.

---



## 19.Emoji表情[":"前置]

```
:sweat_smile:
:drooling_face:
:clown_face:
//敲回车或鼠标点击,后置的":"一般不需要自己输
```

E.G.

:sweat_smile:
:drooling_face:
:clown_face:

对于其余普通的Markdown文本编辑器,可以直接将Emoji表情复制进来,这是直接**硬编码**的(~~刻进DNA里~~)

用好这个功能可使文本非常~~可爱~~抽象
